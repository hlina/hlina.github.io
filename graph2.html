Linear Program Variables Explained

Objective function: minimize traveling distances of all delivery trucks
yij is a cost matrix parameter and Xijv is a binary variable representing whether the arc ij is traveled and thus whether the cost yij is incurred. Parameters:
• dj and pj represent the delivery and pickup request values at station j
• Q is vehicle capacity, which is 25 bikes
• yij is the cost of travel between stations i and j
This is estimated based on the distance matrix, which is in kilometers, and the cost of gas in traveling such a distance in NY (estimated to be about $0.379)
• V is the total number of vehicles.
It was decided to use 2 because 2 vehicles can carry in total 50 bikes. There are 10 demand stations which need 5 bikes each. These are arbitrary assumptions that require more research
• TL is max route for any given vehicle
This was made infinite for now, but will be researched in depth later.
Variables:
• Xijv is a binary variable that represents whether the path between station i and station j is taken for vehicle v.
• Div number of deliveries yet to be made and Piv represents the cumulative number of bikes picked up at station i on vehicle v
Constraints Explained:
8) This constraint makes sure each node is traveled to once (since j is the node the truck arrives at)
9) This constraint makes sure each vehicle is used at most once and thus leaves the starting station only once. (In our GAMs code, the starting station is station 1.)
10) This ensures that a vehicle that arrives at a node must depart from that node as well.
11) This ensures that for every node that the vehicle leaves, the load the vehicle carries is smaller than the vehicle capacity.
12) This ensures that at the starting node, the total number of bikes yet to be delivered is equal to the total demand at all the stations the vehicle will travel to.
13) This ensures that at the starting node, there have been 0 pickups.
14) This ensures that if station j is visited, Dj or number of bikes yet to be delivered at the new station is equal to Di, the number of bikes to be delivered at the last station the vehicle just left from, minus the amount of bikes delivered at station j. This is a non-linear constraint.
15) This ensures that if station j is visited, Pi, or number of bikes cumulatively that have been picked up, increases by pj, for the new total number of bikes cumulatively picked up at station j. This is also a non-linear constraint.
16) This is to ensure the maximum route traveled is less than the maximum route for a vehicle, but since this is infinite in our problem, this constraint really isn’t necessary.
17, 18, 19) These constraints stipulate that Piv and Div are positive variables and Xijv is a binary variable, which are the types that make sense for what they represent.
